# Features

Core features of the Blood Cell Classification System.

---

## 1. Real-Time Image Classification

Upload blood cell image and get instant predictions.

- Classify 4 cell types (Lymphocyte, Monocyte, Neutrophil, Eosinophil)
- Process time: <500ms
- 72.98% accuracy on test dataset
- Support JPG, PNG (max 10 MB)
- Automatic resizing for any resolution

---

## 2. Confidence Score

View probability estimates for predictions.

- Confidence percentage display
- Per-class probability breakdown
- Well-calibrated confidence levels

---

## 3. Per-Class Probability Analysis

Detailed probability distribution across all classes.

- Bar chart visualization
- Numeric percentages
- Color-coded classes

---

## 4. Drag & Drop Upload

Simple image upload interface.

- Click to select or drag & drop
- File preview before upload
- Progress indication
- Error messages

---

## 5. Image Preview

See original and preprocessed images.

- Original image preview
- Preprocessed version (model input)
- Side-by-side comparison
- Image metadata display

---

## 6. Results Display

Prediction results presentation.

- Predicted class name
- Confidence percentage
- Class probabilities
- Processing time

---

## 7. Responsive Design

Mobile-friendly interface.

- Desktop support
- Tablet support
- Mobile support
- Dark mode
- Touch-friendly

---

## 8. Prediction History

Access past predictions.

- Sortable list with timestamps
- Filter by class type
- Search by image name
- Pagination support
- Delete individual items

---

## 9. Prediction Details

View detailed information.

- Original image
- Preprocessed image
- All probability scores
- Processing metadata
- Timestamp and model version

---

## 10. Clear History

Manage stored predictions.

- Delete single prediction
- Delete multiple selections
- Clear all history
- Confirmation dialog

---

## 11. Statistics Dashboard

Analytics and metrics.

- Total predictions count
- Success rate
- Average confidence
- Class distribution
- Processing speed stats
